# Kung Fu Shaolin Norte

Aplicação frontend criada com Vue 3 e Vite para organizar treinos de kung fu Shaolin do Norte.

## Como usar

1. Instale dependências:
   ```bash
   npm install
   ```
2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## Estrutura

- `src/main.js` - ponto de entrada da aplicação
- `src/App.vue` - layout principal
- `src/components/TrainingCard.vue` - cartão de planos de treino
- `src/assets/styles.css` - estilo global
